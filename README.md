#!/Aplikasi Pencatat Keuangan

Aplikasi web untuk menyimpan data akun (link, username, password) beserta catatan kemenangan dan kekalahan. Dilengkapi dengan grafik statistik dan fitur export.

## Fitur
- Simpan data akun (link, username, password)
- Catat kemenangan & kekalahan
- Grafik statistik (global & per akun)
- Pencarian cepat
- Export data
- Dark mode
- Responsive

Tech: Next.js 16 + Drizzle ORM + PostgreSQL + Tailwind CSS
